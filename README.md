# StockTradingManagement
Stock Trading Management System


V1.0
 - 기본 기능 구현
 
V1.01
 - 주식 이름과 매입가를 수정할 수 있도록 변경했습니다.
 - 인터넷을 통해 현재가를 불러올 수 있도록 종목 코드 입력 기능을 추가했습니다.
 - 종목 코드 정보를 이용하여 현재가를 불러와서 업데이트할 수 있는 기능을 추가했습니다.

V1.02
 - 버튼 이름 변경: index.html 파일의 '매매 설정' 버튼 텍스트를 **'설정'**으로 변경했습니다.
 - 환율 입력 필드 추가: 설정 섹션에 '환율' 입력 필드를 추가했습니다.
 - 환율 정보 표시: 화면 상단에 매도 비율, 추가 매수 비율과 함께 현재 설정된 환율 정보가 표시됩니다.
 - 가격 정보 환산: 입력된 환율 값에 따라 매입가, 현재가, 고점, 매도/추가 매수 가격 등 모든 가격 정보를 원화로 변환하여 표시하도록 했습니다.
