주식 매매 관리 프로그램 

처음에 가볍게 생각을 하고 만들었는데, 자신의 주식 정보를 입력하여 실제로 관리하는 데에 도움이 되도록 전면적인 개편을 시작한다.

요구사항 - 일단 되는대로 작성하고, 나중에 분류를 해보자...

* 웹 기반 GUI를 가진 프로그램
* 내부 알고리즘은 Python으로 작성
* 설정 기능
  * 환율 설정 기능
  * 매도 기준 1 단계 (기본 4% 하락) - 하락 알람
  * 매도 기준 2 단계 (기본 8% 하락) - 결정을 내려야 하는 타이밍
  * 업데이트 주기 설정 (기본 10분 단위)
* 초기 동작
  * 현재 환율 / 현재가 정보를 불러와서 업데이트
* 입력 정보
  * 종목명 / 종목 코드 / 매입가 / 수량
* 테이블 정보
  * 종목명 / 종목 코드 / 매입가 / 수량 / 현재가 / 고점 / 변동 / 매도 / 손익 / 의견 / 관리
    * 종목명 - 테이블에서 수정 가능 
    * 종목 코드 - 테이블에서 수정 가능 / 현재가 검색에 사용되는 정보
    * 매입가 - 테이블에서 수정 가능
    * 수량 - 테이블에서 수정 가능
    * 현재가 - 매입가와 동일한 값으로 시작하며, 업데이트되는 정보
    * 고점 - 매입가 또는 현재가 중에 높은 값으로 계속 업데이트 됨
    * 변동 - 고점 기준 매도 1단계 금액 업데이트
    * 매도 기준 2 단계 - 고점 기준 매도 2단계 금액 업데이트

  * 삭제
  * 매수 기록
  * 매도 기록
* 알림 기능
